51       * Enum used to indicate what each player is allowed to spend life on.
52       * By default it is set to `allAbilities`, but can be changed by effects.
53       * E.g. Angel of Jubilation sets it to `nonSpellnonActivatedAbilities`,
54       * and Karn's Sylex sets it to `onlyManaAbilities`.
55       * <p>
56       * <p>
57       * Default is PayLifeCostLevel.allAbilities.
58       */
59      enum PayLifeCostLevel {
60          allAbilities,
61          nonSpellnonActivatedAbilities,
62          onlyManaAbilities,
63          none
64      }

72       * @return
73       */

79       * Current player is AI. Use it in card's code and all other places.
80       * <p>
81       * It help to split Human/AI logic and test both by unit tests.
82       * <p>
83       * Usage example: AI hint to skip or auto-calculate choices instead call of real choose dialogs
84       * - unit tests for Human logic: call normal commands
85       * - unit tests for AI logic: call aiXXX commands
86       *
87       * @return
88       */
